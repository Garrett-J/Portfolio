Dice Roll Game

When:

  • January 2022

Goals:

  • Customizable names
  • Custom winner's message
  • Display dice images that correspond with numbers generated

Tools Used:

  • HTML5, CSS3, JavaScript, Bootstrap

Issues Faced

I needed to be able to randomly generate two different numbers between 1 and 6 and use those two numbers to change their images and declare the winner. I had the variables randomNumber1 and randomNumber2 that were set to change the image and text display based on their values, but I was having difficulties generating the two numbers to be random and within my range.

Solutions Found

I ended up learning and putting into practice the Math.random() function. Using Math.floor() before it, I was able to make sure that it only returned whole numbers. My code then looked like Math.floor(Math.random() *6); This wasn’t working as it was returning 0’s back, which did not have a corresponding dice face for. I needed to set a starting number, which was 1, so that it started the count at 1 and generated numbers up to 6. The final function ended up looking like this: 

function rollDice() {
    var randomNumber1 = Math.floor(Math.random() * 6) + 1;
    var randomNumber2 = Math.floor(Math.random() * 6) + 1;
}
