Geoff Morgan Photography

Goals

I created the website for Geoff Morgan Photography in May of 2022 working one on one with the client from start of the design process to the publishing of the website. The site is comprised of a landing page, and a portfolio page containing a responsive image gallery. The three most important things for the client were mobile responsiveness, a simple layout and being able to update content in the future. To achieve these three goals set out by the client I developed a Wordpress theme with PHP that makes use of SASS media queries in combination with flexbox layouts to achieve functional layouts on all device sizes. Custom fields were implemented in order for the client to be able to easily update all content in an easy-to-understand manner. This being my first project for a client, understanding the Wordpress interface and being able to create something that will actually by customers gave me numerous ways to further my understanding of working with a client and putting their ideas out into the real world in a way that will benefit them and their product.

Issues Faced

The client wanted a portfolio page that would display any assortment of images they chose all at once in a seamless fashion while adapting to any screen size without drastically changing the size of the images or adding empty space on the sides of the page.

Solutions Found

Then to add the responsiveness, I added CSS that declared the number of columns that should appear depending on the screen size. The larger the screen size the more columns will be displayed, and therefore more pictures will be displayed side by side.


.row {
    line-height: 0;
    column-count: 1;
    column-gap: 5px;

    @include media-md {
        column-count: 2;
    }

    @include media-lg {
      column-count: 3;
    }

    @include media-xl {
        column-count: 4;
    }

    .row-img {
        width: 100%;
        height: auto;
        margin-bottom: 5px;
    }
}
