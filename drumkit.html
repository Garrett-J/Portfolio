Drum Kit

Goals

I created the Drum Kit website in February of 2022. The aim of the project was to create a website that displayed a playable drumkit. I build the website using HTML and CSS, and used JavaScript to create the functionality I wanted out of the drumkit. It was important to me that I allow both desktop and mobile users to use the drums in whatever way they prefer, so I made sure that the buttons can be activated using either a key on the keyboard or directly clicked.

Issues Faced

I was testing the site when the kit suddenly wasn’t responding to my keypresses! I realized that it was due to my Caps Lock being on. Since I hadn’t added anything telling the user to disable their Caps Lock key in order for the site to work, I needed to figure out how to make everything function the same whether or not Caps Lock is on or off. My accidental use of Caps Lock showed me the importance of thorough bug testing, since you never know what might end up breaking your website.

Solutions Found

Here were three changes that I ended up making to solve the problem:

I added a second case which included the capitalized letter. This allowed the audio to play if either the corresponding lowercase or uppercase key was pressed. 


function makeSound(key) {
    switch (key) {
        case "W":
        case "w":
            var tom1 = new Audio("sounds/tom-1.mp3");
            tom1.play();
            break;
            
        case "A":
        case "a":
            var tom2 = new Audio("sounds/tom-2.mp3");
            tom2.play();
            break;

Now that the sound plays when Caps Lock is on, I need the pressed class to be added and removed with the buttonAnimation function. To achieve this, I added another class to each HTML element which was the letter, but capitalized.


<button class="w W drum">w</button>
<button class="a A drum">a</button>
<button class="s S drum">s</button>
<button class="d D drum">d</button>
<button class="j J drum">j</button>
<button class="k K drum">k</button>
<button class="l L drum">l</button>

To make the function select the element when it was pressed, I added , currentKey to “.” + currentkey. This assigned the element to the variable activeButton whether it was a capitalized or not 


function buttonAnimation(currentKey) {
    var activeButton = document.querySelector("." + currentKey, currentKey);
          
    activeButton.classList.add("pressed");
          
    setTimeout(function () {
        activeButton.classList.remove("pressed");
        }, 100);
    }
