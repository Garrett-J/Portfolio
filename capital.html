Capital Restaurant

Mobile

    In August of 2021, I began a personal project of redesigning and building a local restaurant’s website to work on improving my coding skills.

    I looked at their current website, and made some basic notes of things I’d like to improve or change. I wanted the menu to be easy to navigate as it is very long and in it’s current state it’s very difficult to navigate. I also wanted to keep the same website structure, so I set limitations that I had to keep the same pages in my redesign.

    I used the same assets and images that exist in the current website in creating my redesign. With those assets, I rebuilt each of the pages and made sure they were all fully responsive and all of the dynamic features functioned correctly. Some key features that I made sure to include were a contact form, food menu and embedded Youtube videos.

    I published the website using GitHub pages. It is fully functional except for the contact form, which will not send an email as it was built as a personal project, not for a client.
