Capital Restaurant

When:

  • October 2022

Goals:

  • Provide customers with menu, contact information and the restaurant's apperances in media.
  • Redesign current website to improve accessibility.

Tools Used:

  • HTML5, CSS3, JavaScript, jQuery, GIT

Issues Faced

I wanted to break the menu page up into sections since the menu is so large that displaying it all on one page was a lot to navigate for a user, especially on a mobile device. To achieve this, I needed to create a series of responsive buttons that would let the user access one section at a time.

Solutions Found

To create the responsive buttons, I used the CSS display: flex; and flex-flow: row wrap. This allowed the buttons to be completly responsive.

On tablet and mobile, I needed to change it up since displaying all of the buttons on a small screen would mean that it would take up way more vertical space than I would prefer. To turn the buttons into something that scrolled, I turned the button container into a grid that changed the amount of columns displayed based on the size of screen using grid-template-columns
