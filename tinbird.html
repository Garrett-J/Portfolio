Tinbird

Mobile

Goals:

In January of 2022, I created a project page to improve my HTML and CSS. It is a landing page for a fictional company, TinBird, who developed an app similar to Tinder, except for parrots.

I made sure the landing page had information that they would want a customer to see, such as reviews, pricing information and download buttons. I also wanted to make sure the website was responsive as many people would likely be viewing the website on their phone, and then downloading the app.

I created a lofi-mockup of the website layout, and then started the development process. I used Bootstrap to quickly build the responsive layout. I also used Bootstrap JavaScript elements, like the review carousel, to create some dynamic features. I kept the client’s needs in mind in focusing on the responsiveness and the overall professional look that a customer would expect in visiting the website of a well-supported app such as Tinder.

I published the website using GitHub pages. The TinBird website served as both practice in designing and building a website using HTML and CSS and imagining what a client’s needs may be and which ones to prioritize in building a website.
