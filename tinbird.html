Solutions Found

I first tried making them all the same height by adding the class h-100 to each card element. This Bootstrap class sets each card to 100% of the row height. While it did make them all the same height, the "Sign Up" buttons were still out of line:

I removed the class and tried just adding <p>&nbsp;</p> before the Sign Up button for the first two cards. This resulted in all cards being the same height, and the buttons staying at the bottom of the card.
