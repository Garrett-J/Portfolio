Tinbird

Goals

I built TinBird, a landing page for a made-up company which developed Tinder except for parrots, in January of 2022. The website is a landing page designed to encourage people to download the app as well as sign up for higher pricing tiers. The main goal of the website was to create a functionable website that is fully responsive. I used Bootstrap to create the outline for a lot of the sections, which allowed me to easily customize the responsiveness of the website. I used HTML and CSS to build and style the rest, while using built in to Bootstrap JavaScript elements, like the review carousel, to create some more dynamic features. I used this project as a way to understand developing a website for both a potential client and to create a focus on mobile friendly design using Bootstrap and CSS.
