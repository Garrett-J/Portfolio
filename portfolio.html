Portfolio Site

When:

  • June 2022

Goals:

  • Share contact info, resume, and portfolio.
  • Minimalistic & responsive design.

Tools Used:

  • HTML5, CSS3, SASS, jQuery, JavaScript, GIT

Issues Faced

When creating the containers on the landing page that contain each projectâ€™s name, description, information and links, I wanted them to be fully responsive and also use the space well on mobile and desktop. I first styled the containers as flex containers for mobile. Using the column-oriented layout, I was able to position things exactly how I wanted them. When trying to turn this flex-container into a desktop-oriented layout, I ran into issues with being able to control things exactly how I wanted them without re-writing my HTML and in turn, ruining my mobile styling.

Solutions Found

I decided to turn the flex container on mobile screens into a grid container for larger screens. I gave each part in the container a grid-area, and then used grid-template-areas to arrange them. This let me define what I wanted where, in terms of columns and rows. All that I had to do after was adjust the padding on each element individually to get the exact positioning that I wanted.

grid-template-areas:
                    'header header header right right right'
                    'main main main right right right'
                    'left left left right right right'
                    'footer footer footer footer footer footer';

Using a SASS mixin, my separate stylings for the containers ended up looking like this:

.project {
                    display: flex;
                    flex-direction: column;
                    border-radius: 20px;
                    overflow: hidden;
                    width: 95%;
                    border-color: #2CBF6C;
                    border-width: 1px;
                    border-style: solid;
                    align-items: center;
                
                    @include media-lg {
                        display: grid;
                        grid-template-areas:
                        'header header header right right right'
                        'main main main right right right'
                        'left left left right right right'
                        'footer footer footer footer footer footer';
                        padding: 3rem;
                        border-width: 3px;
                        width: 70%;
                        align-items: start;
                    }
                }
