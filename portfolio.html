Portfolio Site

Mobile

In June of 2022, I began work on developing a portfolio website to display my front-end development work.

There were a few things that were important to me in creating the portfolio website. I wanted it to be minimalistic so that employers would be able to quickly find my work, resume and contact information. While keeping it minimalistic, I wanted to give it something that was unique to me.

I created a rotating greeting message that shows a bit of who I am, my interests and personality traits using JavaScript. Front and center upon loading the page are my contact information as well as a link to my full resume. When scrolling down, I built fully responsive sections containing each project and info cards showing the tools used to build them. Each project contains both a link to the live website as well as a separate page containing a responsive demo and a small writeup to share some basic information outlining the project. The portfolio website also contains a viewable and downloadable resume which is important for possible employers to be able to quickly access.

After publishing the website, I am able to easily add new projects and work to my portfolio where they can be easily accessible by future clients and employers.
